# Adaptive diagnostic — clickable demos

Two standalone HTML prototypes for the adaptive reteach system. No build step,
no server — open directly in a browser.

- `diagnostic-prototype.html` — generic math example (fractions/LCM).
- `diagnostic-demo.html` — seeded with a real student's Language deep-dive
  data (name, campus, and guide removed).

Full system logic: see the `system-logic.md` note in the source project.
